Instructor Links Help Page

Instructor Tools
Quick access to many instructor tools, including Reset passwords and Act as user (student). Beginners may find it easier to use the "Class List Editor", "Hmwk Sets Editor" and "Library Browser" at first.
Class List Editor
Edit class roster data. Add students, edit student data, drop students from class, import students from a classlist, and give user professor privileges. To reset passwords use "Instructor Tools"
Hmwk Sets Editor
Change homework set due dates, create new sets from a set definition file, create new sets, make sets visible/invisible, score homework sets.
Library Browser
Choose problems from a library and add them to a homework set.
Set Assigner
Assign homework sets to students.
Statistics
View statistics of students' performance on homework.
Student Progress
Progress for individual students
Scoring Tools
Score one or more sets. This can also be done from the "Hmwk Sets Editor"
Email
Send email to students.
File Transfer
Upload, download and delete text files, including scoring spread sheets, set definition files, class list spread sheets, and "PG" problems.
